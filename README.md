# Hoe Paxel

Very simple mod that adds paxels and combines sword, spear, axe, pickaxe, shovel, and hoe into one tool.  
Nothing else, so if you want fancy tweaks, check out other mods.

## Usage

Craft a paxel by placing the sword, axe, pickaxe, shovel, and hoe of the same material in a crafting table in any order.  
Wood, stone, copper, iron, gold, diamond, and netherite are supported.

Press `H` to switch the main-hand right-click mode between shovel and hoe.  
The offhand always uses the opposite mode, and the active shovel or hoe is rotated in the item sprite.

Press `J` to switch the weapon mode between sword and spear.  
Switch to sword mode when breaking blocks.
Mode changes update immediately and show a toast notification.
